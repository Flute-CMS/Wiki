# Wiki
База знаний для вашего проекта. Модуль позволяет создавать справочные статьи и организовывать их по категориям. Каждая статья поддерживает Markdown-разметку, автоматическое оглавление по заголовкам, теги для удобного поиска и систему оценки полезности материала.  На главной странице базы знаний отображаются все категории со списком статей внутри. В
